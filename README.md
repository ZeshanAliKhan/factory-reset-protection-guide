# Factory Reset Protection Guide

Static GitHub Pages site about safe recovery after Android Factory Reset Protection locks a phone, with emphasis on account recovery, ownership checks, and avoiding shady bypass APKs.

Suggested live URL after GitHub Pages is enabled:

https://zeshanalikhan.github.io/factory-reset-protection-guide/

## Included

- Homepage with safety-first FRP explainer
- Supporting pages for why FRP exists, recovery steps, used-phone and managed-device cases, and FAQ
- GitHub Pages essentials: `index.html`, `404.html`, `robots.txt`, `sitemap.xml`, `.nojekyll`
- Reused cover image asset from the prepared Vocal article package
